# ⚡ ML-STA Engine

## Physics-Informed Machine Learning for Static Timing Analysis

A Physics-Informed Machine Learning framework for SPICE-accurate CMOS gate delay prediction and circuit-level Static Timing Analysis (STA).

### Highlights
- 0.82% error vs SPICE on C17 benchmark
- Up to 49× estimated speedup on 432-gate benchmark
- Physics-informed feature engineering
- FT-Transformer + PINN concepts
- Custom ML-based STA propagation engine
